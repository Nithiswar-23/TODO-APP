# TODO APP (Income Expense Calculator)

## 📌 Description
This project is a fully functional Income and Expense Tracker built using HTML, CSS, and JavaScript. It enables users to manage financial transactions through CRUD operations and provides a real-time summary of income, expenses, and balance.

## 🚀 Features
- Add, edit, and delete income and expense entries
- Categorization using filters (All, Income, Expense)
- Real-time calculation of total income, expenses, and net balance
- Persistent data storage using Local Storage
- Dynamic rendering of transaction list
- Input reset functionality

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Open `index.html` in a browser
2. Add income or expense entries
3. Manage transactions using edit/delete options
